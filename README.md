🛍️ Alani Clone Website

A responsive Alani Brand Website Clone built using HTML & CSS.
This project replicates the UI, layout, and modern design of an Alani-style e-commerce/wellness brand website.

👉 https://stupendous-cat-afe0fe.netlify.app/

🏢 About Alani

Alani is commonly known as a modern lifestyle and wellness brand offering products like energy drinks, supplements, and fitness-related items. It focuses on clean ingredients, vibrant design, and empowering users toward a healthier lifestyle.
